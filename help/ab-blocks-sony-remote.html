BlocklyProp reference for SONY REMOTE block

For the Propeller Activity Board WX, FLiP, S3 robot, or Other board types. Not available for the Badge board type.

Use this block with a Sony protocol Infrared Remote (#020-00001) and infrared receiver (#350-00039)

New to this sensor? Click here to see an example schematic and quick Blockly program to help you get started.

Sony Remote value

The Sony Remote value block returns a value detected by the IR receiver that indicated which button on the remote was pressed.  If no button press was detected, this block provides a value of -1.
